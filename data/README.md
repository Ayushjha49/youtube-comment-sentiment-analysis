# Data Directory

## Expected format
Place your labeled CSV file at: `processed/comments_cleaned.csv`

### Required columns:
- `comment_text` — raw comment text
- `sentiment` — label: `positive`, `negative`, or `neutral`

### Example:
```csv
comment_text,sentiment
"This video is amazing!",positive
"vayo ni yaar ekdam ramro",positive
"kasto bakwas video ho",negative
"okay video ho neutral lagcha",neutral
```
